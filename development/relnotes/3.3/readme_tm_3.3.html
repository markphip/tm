Target Management Project Release Notes

Release 3.3.1
Last revised Sep 26, 2011

TM Service Release 3.3.1

TM 3.3.1 service release includes 21 bug fixes compared to TM 3.3.
TM 3.3.1 is 100% upward and backward compatible with TM 3.3. If you already have TM 3.3, you can "check for updates" using the Eclipse update manager to get it.

See the TM 3.3.1 Build Notes for details, and this bugzilla query for a list of all defects fixed.

TM Release 3.3

137 unique bugs were addressed in the TM 3.3 release, including 4 API change requests.
See this report for details on all bugs, and this list for new API introduced.

TM 3.3 highlights include a new SCP file subsystem, performance improvements for SSH, IPv6 support for FTP and Telnet, as well as several improvements to the TM Terminal for better ANSI emulation and scrolling.
It is fully compatible with TM 3.2, and runs on Eclipse 3.4 (Ganymede) and later.

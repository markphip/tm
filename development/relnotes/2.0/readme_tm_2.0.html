DSDP Target Management Release Notes

Release 2.0.3
Last revised February 25, 2008

TM Service Release 2.0.3

TM 2.0.3 service release includes 13 bug fixes compared to TM 2.0.2. RSE persistence was made more robust, SSH connections do not time out any more thanks to keepalives, FTP connections can show hidden files, and the Terminal optional input line got Ctrl+Space field assist for quickly accessing its history.
TM 2.0.3 is 100% upward and backward compatible with TM 2.0. If you already have TM 2.0, you can "check for updates" using the Eclipse update manager to get it.

See the TM 2.0.3 Build Notes for details.

TM Service Release 2.0.2

TM 2.0.2 service release includes 24 bug fixes, especially fixing some serious issues with the new fast Terminal implementation, and two problems with file upload.

See the TM 2.0.2 Build Notes for details.

TM Service Release 2.0.1

TM 2.0.1 service release includes more than 150 bug fixes and performance enhancements, particularly

  • Fixes non-ASCII encodings for files and directories on remote FTP or SSH servers
  • includes a lighting fast new Terminal implementation (up to 1000 times faster than the old one on Windows)
  • Several fixes for FTP passive mode, archive handlers and the Eclipse Filesystem (EFS) provider
    • .
EFS works nicely now for mapping remote resources into any kind of Eclipse project.

See the TM 2.0.1 Build Notes for details.

TM Patch Release 2.0.0.1

The TM 2.0.0.1 patch release changes 6 lines of Java code compared to TM 2.0 in order to fix critical bugs 192741 and 194204 which could lead to loss of data when working with ZIP archives or renaming files on an FTP connection. The RSE Core and RSE FTP features were changed. See the TM 2.0.0.1 Build Notes for details.

TM Release 2.0.0

TM 2.0 includes significant cleanups, refactorings and API changes for enhanced flexibility compared to the RSE 1.0 release. 541 unique bugs were adressed, including 136 API change requests. Users will see enhanced FTP configuration options, Terminal improvements, an EFS integration, and Import/Export wizards for bulk uploads and downloads among other improvements.
